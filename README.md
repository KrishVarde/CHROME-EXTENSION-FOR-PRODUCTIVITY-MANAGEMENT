# DESCRIPTION OF TASK PERFORMED : 

As a task of the CODTECH internship, I developed a Productivity Tracking Chrome Extension that helps users monitor their browsing habits, manage distractions, and improve productivity. The primary objective of this project was to create a lightweight browser extension capable of tracking website usage, blocking distracting websites, and generating daily productivity reports.

To build this application, I utilized **HTML**, **CSS**, and **JavaScript** for the extension interface and functionality. The extension was developed using **Chrome Extension Manifest V3**, which provides the necessary APIs for interacting with browser tabs and storing user preferences. On the backend, I used **Node.js** and **Express.js** to create REST APIs, while **MongoDB** was utilized for storing website analytics data and user preferences.

The extension works by monitoring the active browser tab and tracking the amount of time spent on different websites. Using Chrome's tab management APIs, the extension identifies the currently active website and records browsing duration. The collected data is periodically stored and synchronized with the backend database, allowing users to maintain their productivity statistics across sessions.

A key feature of the extension is website blocking. The application includes a predefined list of distracting websites such as YouTube, Instagram, Reddit, and Facebook. Users can also add or remove websites from the blocked list according to their preferences. When a user attempts to access a blocked website, the extension intercepts the request and displays a custom blocked page, preventing access while the blocking feature is enabled.

The extension also generates daily productivity reports based on browsing activity. These reports display total browsing time, time spent on individual websites, and the most frequently visited websites. The collected information is presented through a clean dashboard interface, enabling users to better understand their online behavior and identify productivity patterns.

The user interface was designed to be simple, responsive, and easy to navigate. A popup dashboard provides quick access to browsing statistics, productivity reports, and website blocking settings. User preferences and analytics data are stored efficiently, ensuring a smooth and consistent experience.

Error handling was implemented to manage issues such as failed data synchronization, unavailable backend services, and invalid user inputs. The project follows a modular structure, making the codebase easy to maintain and extend in the future.

Overall, this project demonstrates the integration of Chrome Extension APIs, Node.js, Express.js, and MongoDB to create a practical productivity management tool. The extension provides users with valuable insights into their browsing habits while helping them reduce distractions and maintain focus during their daily activities.
